Medical Shop Website

The Medical Shop Website is a simple web-based platform designed to help customers view available medicines, check prices, and contact the pharmacy online. The website provides a digital interface for a local medical shop to showcase their products and services. Objective: To create an easy-to-use website that allows users to explore medicines and get information about pharmacy services. Technologies Used HTML CSS Basic Web Design Features Homepage with shop introduction Medicine product listing Search or browse medicines Contact information of medical shop Clean and user-friendly interface Benefits Customers can check medicines online Saves time visiting the shop Helps the medical store reach more customers Future Improvements Online medicine ordering Payment gateway integration Doctor prescription upload Login system for customers

Line Detector Robot

he Line Detector Robot is an autonomous robot designed to detect and follow a black line on a white surface. The robot uses sensors to continuously monitor the path and automatically adjusts its direction to stay on the line. This project demonstrates the integration of electronics, sensors, and programming to build an intelligent robotic system. Objective The objective of this project is to design a robot that can automatically follow a predefined path using sensors and a microcontroller. Components Used Microcontroller / Arduino IR Sensors Motor Driver Module DC Motors Robot Chassis Battery
